# AeroCode

Instruções para clonar e rodar o projeto localmente.

### 1. Clonar o repositório
```bash
git clone https://github.com/GustavoMGreco/AV2.git
```

### 2. Instalar dependências
```bash
npm install
```

### 3. Rodar o projeto
```bash
npm run dev
```

O projeto estará disponível em `http://localhost:5173`.
